# 🚀 Bizpilot

**Bizpilot** es una solución SaaS (Software as a Service) diseñada para centralizar y simplificar la administración de negocios. Permite gestionar de manera eficiente el flujo de clientes, inventarios y transacciones comerciales desde una interfaz intuitiva.

## ✨ Funcionalidades Clave

* **Dashboard Operativo:** Resumen visual del estado actual del negocio.
* **Gestión de Clientes:** Control y seguimiento de base de datos de contactos.
* **Catálogo de Productos:** Administración de inventario, precios y stock.
* **Ventas:** Registro histórico y gestión deventas.

## 🛠️ Stack Tecnológico

* **Frontend:** React / Next.js (Desplegado en **Vercel**).
* **Backend:** Python / FastAPI (Hospedado en **Render**).
* **Base de Datos:** PostgreSQL (Gestionada en **Render**).
